Add a vagrant box (.box file)

$ vagrant box add --name [name_of_the_box] [path/url to .box file]

$ vagrant box add --name secure_buildbox ./secure_buildbox.box

...

List vagrant boxes

$ vagrant box list

secure_buildbox (virtualbox, 1.0.0)

...

Initialize vagrant environment

$ mkdir /home/praveen/secure_buildbox

$ cd /home/praveen/secure_buildbox

$ vagrant init secure_buildbox

A `Vagrantfile` has been placed in this directory. You are now
ready to `vagrant up` your first virtual environment! Please read
the comments in the Vagrantfile as well as documentation on
`vagrantup.com` for more information on using Vagrant.

$

Vagrant up!

$ pwd

/home/praveen/secure_buildbox

$ vagrant up

Suspend Saves VM state

$ pwd

/home/praveen/secure_buildbox

$ vagrant suspend

Shutdown Gracefully shutdown VM

$ pwd

/home/praveen/secure_buildbox

$ vagrant halt

Destroying a box

$ pwd

/home/praveen/secure_buildbox

$ vagrant destroy
